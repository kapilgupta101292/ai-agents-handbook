## What is Nextra?

Nextra is a powerful static site generator built on Next.js for creating documentation and blogs with MDX support.

## Features

- 📝 Write in **Markdown** and **MDX**
- ⚡ Fast and optimized performance
- 🎨 Beautiful default theme
- 🔍 Built-in search functionality
- 📱 Fully responsive
